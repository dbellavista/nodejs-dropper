# Dropper node.js (PoC)

Author: Daniele Bellavista

Incomplete dropper in node.js for penetration testing.

## Requirements

The module need to be compiled inside a Windows environment with node.js installed.

For compilation gyp is required and also Visual Studio (Express) and Windows SDK for 64 bit systems (check out gyp installation guide).

    npm install -g gyp

For creating the executable

    npm install -g enclose

## Limitation

Payload allocation and execution is performed by a native addon. Currently the creation of a portable exe embedding a native addon is not supported. The dropper could be used however using a self-extracting archive.

## Compilation

To compile the native addon execute gyp inside a windows environment. Upon successful compilation, the module will be available in `build/Release/dropper.node`.
Copy `dropper.node` inside the main directory.

    cd E:\path\to\nodejs-dropper\
    npm install
    copy .\build\Release\dropper.node .\

## Usage

    node main.js
    enclose --loglevel info -x -o main.exe main.js
    copy main.exe ~\Desktop
    copy dropper.main.exe.node ~\Desktop\dropper.node
    cd ~\Desktop
    main.exe

Of course you can also double click on `main.exe`
